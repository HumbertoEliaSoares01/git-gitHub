  • Como usar Git e GitHub
    da Forma mais Fácil Possível

    Hoje eu quero te mostrar como usar Git e GitHub da maneira mais fácil possível para que você aprenda como fazer o gerenciamento de códigos.

    Eu vou te mostrar como usar GitHub Desktop para que você possa gerenciar códigos para facilitar esse processo e você não ter que salvar tudo só no seu computador.

    Como gerenciar código? Isso vai ser simples, nós vamos utilizar o GitHub para nos ajudar, pois ele é uma ferramenta de gerenciamento de códigos, na verdade gerenciamento e versionamento de códigos.

    Isso quer dizer que você vai poder salvar seus códigos e salvar versões dele, assim sempre que for fazendo alguma atualização você vai ter essas diferentes versões salvas caso precise voltar ou até mesmo fazer algum ajuste.

    Outro ponto importante que vou te mostrar é como criar repositório no GitHub, ou seja, uma “pasta” que vai conter todos os arquivos do seu projeto, assim você vai conseguir de forma fácil e rápida acessar seus códigos e fazer as alterações necessárias.

    E aí, curioso para aprender como fazer o gerenciamento de códigos utilizando o GitHub? Então bora pra aula que tem bastante conteúdo!
